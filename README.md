# Web-Programlama-Odev2
# EDANUR BUTİK - WEB PROGRAMLAMA ÖDEVİ

## PROJE AMACI
Bu projede CSS ile layout (Flexbox ve Grid), responsive tasarım, typography, bileşen stilleri, hover/focus durumları ve CSS değişkenleri kullanımını uygulamalı olarak göstermek amaçlanmıştır.

---

## KULLANILAN LAYOUT YÖNTEMLERİ

### Flexbox
- Header ve filtre satırında Flexbox kullanıldı.
- Menü hizalama sağlandı.
- Chip grubu hizalandı.
- Responsive kırılımda header alt alta geçecek şekilde düzenlendi.

### CSS Grid
- Ürün kartları bölümünde kullanıldı.
- Büyük ekranlarda 4 sütun.
- Tablet boyutunda 2 sütun.
- Mobilde 1 sütun.

---

## RESPONSIVE BREAKPOINTLER

- **1024px ve üzeri:** Ürün grid’i 4 sütun.
- **600px – 1023px:** Ürün grid’i 2 sütun.
- **600px altı:** Ürün grid’i 1 sütun ve header menü alt satıra düşer.

Media query kullanılarak responsive yapı sağlanmıştır.

---

## ÖĞRENDİKLERİM VE ZORLANDIKLARIM

- CSS Grid ile kolon bazlı layout oluşturmayı öğrendim.
- Flexbox ile hizalama ve boşluk kontrolünü geliştirdim.
- Media query ile responsive tasarım mantığını pekiştirdim.
- Hover ve focus durumlarının kullanıcı deneyimi açısından önemini fark ettim.
- Responsive kırılımları ayarlarken grid yapısının bazen beklediğim gibi davranmaması beni biraz zorladı.
- Flexbox ve Grid’i birlikte kullanırken hangi yapıyı nerede kullanmam gerektiğini başta karıştırdım ancak deneme yaparak daha iyi anladım.
